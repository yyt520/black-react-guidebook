# Mobx

Mobx 通过透明的函数响应式编程（transparently applying functional reactive programming - TFRP）使得状态管理变得简单和可扩展。

Mobx 主要有以下几个核心特性：

- Observable State（可观察状态）
- Computed Values（计算属性）
- Reactions（反应，完成页面渲染、日志打印）
- Actions（动作）
